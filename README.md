```

```

---

# вҡҷ Backend Setup

```bash
cd backend
```

### Configure Database

Update:

```
src/main/resources/application.properties
```

Example

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/tempvision
spring.datasource.username=root
spring.datasource.password=your_password
```

Run Backend

```bash
mvn clean install
```

```bash
mvn spring-boot:run
```

Backend runs on

```
http://localhost:8080
```
